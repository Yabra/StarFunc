# STAR FUNC

**Мобильная логическая игра**, в которой игрок восстанавливает разрушенные созвездия, решая задачи на координатной плоскости и работая с графиками математических функций.

## Описание

STAR FUNC — это casual-головоломка с космической тематикой. Игрок выступает в роли космического призрака — последнего хранителя созвездий. Небо разрушено, звёзды утратили связи, и задача игрока — восстановить их, решая уровни с нарастающей сложностью.

Каждый уровень — задача на координатной плоскости: определить закономерность расположения точек, выбрать или построить функцию и сформировать правильный график. Звёзды выступают точками графиков, а результат решения формирует созвездие.

## Основные возможности

- **5 тематических секторов** — от линейных функций до комбинированных зависимостей
- **100 уровней** с постепенным усложнением механик
- **6 типов заданий**: выбор координаты, выбор функции, настройка графика, построение функции, поиск ошибки, восстановление созвездия
- **Система прогрессии** — звёзды, фрагменты, открытие секторов
- **Персонаж-компаньон** — космический призрак с эмоциональными реакциями
- **Сюжетные вставки** между секторами

## Архитектура

Проект построен на **слоёной архитектуре** с разделением ответственности:

```txt
Presentation → Game → Systems → Data → Infrastructure
```

Ключевые принципы:

- **ScriptableObject-driven data** — конфигурация хранится в SO
- **Событийная архитектура** — взаимодействие через SO-события и C# events
- **Сервис-локатор** — глобальные системы доступны через интерфейсы
- **Отделение данных от логики** — конфигурация → сервисы → представление

## Структура проекта

```txt
Assets/
├── Scripts/           # Исходный код (C#)
│   ├── Core/          # Сервис-локатор, события, утилиты
│   ├── Data/          # Модели данных и ScriptableObject
│   ├── Gameplay/      # Координатная плоскость, звёзды, графики, уровни
│   ├── Meta/          # Прогрессия, экономика, жизни, магазин
│   ├── UI/            # Экраны, попапы, виджеты
│   └── Infrastructure/# Сохранения, сцены, аналитика
├── Scenes/            # Boot → Hub ⇄ Level (аддитивная загрузка)
├── ScriptableObjects/ # Данные уровней, секторов, функций
├── Prefabs/           # UI, геймплей, эффекты
├── Art/               # Спрайты, анимации, шейдеры
└── Audio/             # Музыка и SFX
```
